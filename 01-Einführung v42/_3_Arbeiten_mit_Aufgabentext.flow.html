Funktionen im Aufgabentext

Textmarker

Ein häufig genutztes Hilfsmittel, um sich einen Text zu erschließen, ist ein Textmarker. 

In 5Code können Sie ebenfalls einen Textmarker verwenden, um um im Aufgabentext wichtige Stellen zu markieren. Dafür stehen Ihnen 4 Farben zur Verfügung. Sie finden das Textmarker-Tool in der Menüleiste jedes Tabs, der ein Aufgaben-Dokument zeigt:

  • Selektieren Sie Text (mit Maus oder Shift-Cursortaste) und klicken Sie auf das gewünschte Farb-Tool :
  • Klicken Sie auf die gewünschte Farbe, dann färben Sie ab sofort den Text ein, den Sie selektieren. Das Farb-Tool wird als ausgwählt dargestellt: . Klicken sie nochmals auf das Farb-Tool um diese Funktion wieder auszuschalten. 

Außerdem können Sie bereits selektierten Text auf mit Tastatur-shortcuts farbig markieren: (Strg-1,2,3,4).

Mit den 4 Farben können Sie Ihre Markierungen, z.B. so:

  • rot: Fragen oder Probleme
  • gelb: wichtige Begriffe
  • grün: Tipps
  • blau: Konzepte

Oder eine ganz andere Interpretation, das liegt ganz bei Ihnen.

Außerdem können Sie jede farbige Markierung auch mit einer Notiz verknüpfen, das wird im späteren Kapitel Notizen erklärt.

Hinweise und Tipps 

Manchmal braucht man zusätzlich zur Aufgabenstellung noch ein paar zusätzliche Hinweise. Oder man findet keinen Ansatz zur Lösung und braucht einen Tipp. 

Damit Hinweise und Tipps nicht Platz im Text verbrauchen und damit die Übersicht verringern, werden sie zunächst verdeckt. Man kann sie aber mit einem Klick in einem neuen Tab anzeigen. Dort können Sie dann ggf. weitere Informationen aufklappen. 

Sie können/sollten also zunächst versuchen, ohne Hinweis oder Tipp auszukommen. Wenn nötig, bekommen Sie aber diese zusätzliche Hilfestellung. 

Um das Nutzen von Tipps nicht allzu einfach zu machen, und um Sie so zum Selberdenken anzuregen, "kosten" einige Tipps etwas. Die "Währung" in der Sie Tipps bezahlen heißt 5Coin und wurde bereits beschrieben (siehe Kapitel Features und 5Coins).

Konkrete Aufgaben

Die Aufgabentexte bestehen häufig aus viel Text, zur Einführung in die Problemstellung, Hinweise und Tipps (s.o.), Herangehensweise, Abgrenzungen, Ratschlägen, etc. Um da die konkrete Anweisung, was zu tun ist, nicht zu übersehen, werden diese besonders dargestellt.

So sieht es z.B. unten aus:

  • In der Marginalspalte (am linken Rand) gibt es ein erläuterndes Icon
  • Im Text steht die konkrete Aufgabe in auffälliger Schriftform, z.B. fett und hellblau.

Einige Aufgaben-Arten werden im Folgenden kurz erläutert.

Antworten Sie auf eine Frage

Häufig ist eine konkrete Aufgabe eine Frage, für die Sie eine Antwort eingeben sollen, z.B.:

Warum sollte man Code kommentieren?

Antwort: 

-> In dem Fall klicken Sie auf das Fragezeichen, dann wird eine Zeile für die Antwort eingefügt. Diese ist bereits so formatiert, dass Sie ihre eigenen Antworten im Text immer leicht wiederfinden. 

Tun Sie etwas

Wenn Sie z.B. etwas ausprobieren oder wiederholen sollen, dann gibt es eine allgemeine Aufgabe:

Lesen und Verstehen Sie den Absatz oben.

-> Führen Sie die dargestellte Aufgabe aus. 

Codieren Sie eine Lösung

Wie Sie Aufgaben bearbeiten, für deren Lösung Sie Code erstellen sollen, wird später erklärt (Kapitel Codieren in 5Code). 

Eigene Bewertung der Schwierigkeit mit Smileys

Wenn Sie die Maus über das Aufgabe-Icon bewegen erscheint außerdem ein kleines Pop-Up-Menü:

     

Hier können Sie bewerten, wie gut Sie selber mit der Aufgabe zurecht gekommen sind

  • rot: Diese Aufgabe konnte ich nicht (gut) lösen
  • gelb: Bei dieser Aufgabe hatte ich einige Schwierigkeiten
  • grün: Diese Aufgabe konnte ich (sehr) gut lösen

Wenn Sie eine Bewertung ausgewählt haben, wird das entsprechende Icon neben dem Aufgaben-Icon angezeigt:

Auf diese Art und Weise finden Sie sehr schnell die rot und gelb markierten Aufgaben, bei denen Sie

  • noch Fragen haben
  • noch mehr lernen und verstehen sollten

Das hilft Ihnen beim Lernen, bei der Vorbereitung auf die Prüfung und auch im Praktikums-Termin.

Bearbeiten von einfachen Zeichnungen

Wenn, in späteren Terminen, Daten und ihre Zusammenhänge oder Abläufe beschrieben werden sollen, geht das häufig am anschaulichsten mit einfachen Zeichnungen.

Elemente und Verbindungen

Aus dem Menü am linken Rand können Sie die dargestellten Elemente auf die Zeichenfläche ziehen. Der "Speicher-Block" ist ein spezielles Element für OOP1, außerdem gibt es Elemente in den Abteilungen Allgemein, Flussdiagramm und Container. Diese Abteilungen können Sie auf/zuklappen, um die Übersicht zu behalten.  

Für Verbindungen (Pfeile) zwischen Elementen bewegen Sie den Cursor auf ein Element. Von den dann erscheinenden Andockstellen (kleinen grünen Kreisen) können Sie eine Verbindung zu den Andockstellen eines anderen Elements ziehen. 

Zeichenfläche vergrößern

Wenn Sie mehr Platz für Ihre Zeichnung brauchen, ziehen Sie einfach ein Element über den Rand hinaus. Die Zeichenfläche wird automatisch ergänzt.

Zeichnungs-Editor verlassen

Klicken Sie wieder in dieses Dokument hier, um den Zeichnungs-Editor zu verlassen. Die geänderte Zeichnung wird hier angezeigt. 

Sie brauchen Ihre Änderungen nicht zu speichern, das geschieht automatisch.
