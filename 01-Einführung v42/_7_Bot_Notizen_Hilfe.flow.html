Bot

Der Bot bietet folgende Funktionen:

  • Fragen stellen - und Antworten bekommen
  • Anzeige von System-Meldungen
  • Anzeige und Erklärung von Compiler-(Fehler-)Meldungen

Diese Funktionen werden nun im Einzelnen beschrieben.

Fragen stellen 

"Bots" (Abkürzung von "Roboter") kennen Sie wahrscheinlich aus Messenger-Apps oder Webseiten. Der Bot in 5Code kann einfache Fragen beantworten. Sind Sie bei Begriffen der Objektorientierten Programmierung nicht sicher und möchten diese kurz nachschlagen, ist dies über den Bot möglich:

Öffnen Sie den Tab "Bot" in der Sidebar und Sie einen Begriff, z.B.:

Wird ein Begriff gefunden, wird im Chat eine Erklärung angezeigt. Werden mehr als 3 passende Einträge gefunden, liefert der Chat-Bot eine Liste der gefundenen Begriffe.

Über den Link "Alles anzeigen" können Sie den gesamten Erklärungstext ansehen:

Der gesamte Erklärungstext beinhaltet:

  • Alternativbegriffe: Worte, die das gleiche bedeuten, wie der Haupt-Begriff
    • Im Beispiel hatten Sie nach Maschinencode gesucht, der als Alternativbegriff zum Haupt-Begriff gefunden wird.
  • Wenn es sinnvoll ist, gibt es Beispiele, auch Beispiel-Code
  • Verwandte Begriffe werden als "siehe auch:"-Link angeboten. Mit einem Klick kommen Sie direkt zu deren Erklärung.
Sie können jeden Post löschen, um die Übersicht zu behalten.
 
Sie können eigene Anfragen auch wiederholen, d.h. die Frage noch einmal stellen.
 

 

Glossar

Alle Erklärungen, die der Chat-Bot liefert, stehen in einem sogenannten Glossar. Das ist allgemein eine Sammlung von kurzen Erklärungen von Begriffen, Worten, Dingen, Konzepten, Bezeichnungen, etc. 

Wenn Sie Ihren Workspace mit einem Download-Ordner verbunden haben, stehen das Glossar und damit die Erklärungstexte zur Verfügung. Das erkennen Sie an dem Icon unten rechts in der Statusleiste: . Wenn Sie darauf klicken bekommen Sie als Hilfe-Text die Beschreibung aus diesem Kapitel. 

 

System-Meldungen

Neben der Suche nach Begriffen bietet der Chat auch eine Übersicht über wichtige System-Meldungen, wie z.B. Informationen über freigeschaltete Features. Damit können Sie relevante Informationen einfach wiederfinden.

Wie auch bei der Suche nach Begriffen gibt es einen weiterführenden Link zu mehr Informationen.

Notizen

Das zweite Feature der Sidebar sind die eigenen Notizen. Hier können Sie mehrere Notizzettel anlegen und mit beliebigem Text-Inhalt füllen, wie Fragen zur Aufgabenstellung oder TODOs. Damit können Sie Ihre Gedanken, Anmerkungen und Fragen direkt in der IDE 5Code festhalten. 

Eigene Notiz erstellen

Notizen können Sie über das + Icon hinzufügen. 

Schreiben Sie eine eigene Notiz.

Außerdem können Sie Notizen zu einer markierten Textstelle schreiben. Wie das geht, wird im Folgenden beschrieben.  

Text-Markierungen und Notizen verknüpfen

Sie haben bereits das Feature "Textmarker" im Kapitel 2 kennengelernt. Nun können Sie zu jeder farbigen Textmarker-Markierung im Aufgabentext eine zugehörige Notiz erstellen. So können Sie eigene Gedanken, Ideen, Frage, etc. direkt an diese Markierung knüpfen. 

Um eine Notiz zu einer markierten Textstelle zu erstellen, rufen Sie das Kontextmenü irgendwo in der Markierung auf und wählen "Notiz zur Textmarker-Markierung erstellen". Dadurch erscheint im Notiz-Feld eine neue Notiz. Darin ist bereits die Textmarkierung als Referenz enthalten: . Geht man mit der Maus auf die Notiz, wird ein Pfeil mit einem Link zurück zur Textmarkierung angezeigt. Mit einem Klick kommen Sie also von der Notiz zurück zum Markierung im Text. 

Außerdem wird die Textmarkierung mit einem Link zur Notiz ergänzt. Das wird, wie üblich, durch einen Unterstrich dargestellt, wie oben zu sehen. Mit einem Klick darauf kommen Sie zur Notiz.

Erstellen Sie jetzt eine Markierung im Aufgabentext und schreiben eine Notiz dazu.

Probieren Sie dann die Verlinkung in beide Richtungen aus: Textmarkierung <- -> Notiz.

 

Die Farbe der Markierung wird für die Notiz übernommen

Da die Farbe der Markierung für die Notiz übernommen wird, können Sie Ihre Notizen strukturieren. Z.B. so:

  • rot: Fragen oder Probleme
  • gelb: wichtige Begriffe
  • grün: Tipps 
  • blau: Konzepte

Oder eine ganz andere Interpretation, das liegt ganz bei Ihnen.

Erstellen Sie jetzt eine Markierung im Aufgabentext mit einer anderen Farbe und schreiben eine Notiz dazu.

-> Die Farbe der Notiz hat sich entsprechend angepasst.

Übrigens: Nachträglich können Sie die Farbe einer Textmarkierung nicht mehr ändern. Sie können die Farbe aber entfernen (mit dem Icon ), die Notiz bleibt dabei erhalten. Nur die Verlinkung ist damit entfernt. 

 

Notiz löschen

Sie können einzelne Notizen auch wieder löschen, um z.B. die Übersicht zu verbessern. Dazu erscheint ein Button , wenn Sie die Maus auf die Notiz bewegen.

Allerdings kann man das Löschen nicht wieder rückgängig machen!
