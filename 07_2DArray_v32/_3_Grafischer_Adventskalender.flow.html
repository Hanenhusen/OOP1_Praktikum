Optional: Grafischer Adventskalender

 -------------------

Diese Aufgabe ist komplett optional.

Bearbeiten Sie sie aber gerne, um weiter zu üben (immer gut!) und um schon mal einen Ausblick auf eine einfache grafische Ausgabemöglichkeit zu bekommen.

 ------------------

Nachdem der Adventskalender auf der Textkonsole nun funktioniert, wollen wir diesen so ergänzen: 

  • Die Türen werden grafisch dargestellt 
  • Die Türen kann mit einem Mausklick öffnen

 

Grafik-Konsole verwenden

Für die grafischen Elemente wird die Grafik-Konsole verwendet. Diese wird rechts neben der bisher verwendeten Text-Konsole angezeigt.

 

Grafik-Konsole initialisieren

Damit die neuen grafischen Möglichkeiten genutzt werden können, muss die Grafik-Konsole bereit gemacht, d.h. initialisiert werden. Das geschieht mit dem Aufruf GrafikKonsole.clearGrafik().

Um die im Folgenden erstellten Prozeduren testen zu können, beginnt die Startprozedur main() mit dieser Anweisung.

Der Reaktions-String eines Buttons entspricht dem Eintippen eines Textes oder einer Zahl in die Text-Konsole.
 

 

Auch grafische Türen in einem Array speichern

Bisher haben wir ein 2D-Array von Zahlen verwendet, um die Türen zu speichern. Nun wird jede Tür wird außerdem grafisch dargestellt. Das geht ganz einfach mit einem Button-Grafikobjekt. Sie nutzen dafür den neuen Datentyp Button, der zum Bereich der Grafik-Konsole gehört. Button ist, wie Array und String auch, ein Referenzdatentyp.

Diese Art von Türen muss natürlich ebenfalls gespeichert werden, um mit ihnen im Programm arbeiten zu können. Zum Speichern wird daher ein weiteres 2D-Array verwendet, diesmal als 2D-Array von Button.

Zum Testen der neuen Prozeduren, definieren Sie nun eine entsprechende 2D-Array-Variable  in main(), die auch sofort mit einen Array der passenden Größe initialisiert wird

("initialisieren" heißt hier: Es wird nur ein 2D-Array der richtigen Größe erzeugt. Dieses Array wird hier noch nicht mit Werten besetzt, das machen wir im nächsten Schritt).

Türen auch grafisch anzeigen

Die Türen auch grafisch anzuzeigen ist einfach: Bei der Definition eines Button-Grafikobjekts wird es bereits auf der Konsole angezeigt. Da Sie sie gerade schon erzeugt (und im Array gespeichert) haben, brauchen Sie hier nichts ergänzen.

 

Türe auch grafisch öffnen

Beim Öffnen der Türe soll nun auch noch der entsprechende Button von der Grafik-Konsole entfernt werden. Das muss zusätzlich zu den bisherigen Aktionen (Suchen) gemacht werden.
