Schleifen

Sie lernen in dieser Aufgabe die verschiedenen Arten von Schleifen in Java kennen.

Dazu verwenden Sie zunächst sehr einfache Beispiele, in denen Sie nur Reihen von Zahlen auf verschiedene Arten und Weisen erzeugen.

Da die Beispiele sehr einfach sind, können Sie sich auf den reinen Umgang mit Schleifen konzentrieren. In der nächsten Aufgabe wenden Sie dann Schleifen auch an.

 

Einfache Schleifen

Geben Sie in einer Schleife alle Zahlen n von 1 bis 10 (jeweils einschließlich) auf die Konsole aus.

Implementieren Sie diese Schleife als while-Schleife im Code-Rahmen. 

 

Andere Schrittweiten

Geben Sie nun in einer Schleife alle Zahlen n von 20 bis 11 (jeweils einschließlich) in Schritten von 3 aus. Dazu verwenden Sie unterschiedliche Schleifen-Arten

 

for-Schleife mit Schrittweite

Implementieren Sie diese Schleife im Code-Rahmen als for-Schleife

Testen Sie diesen Code jeweils mit verschiedenen (kleinen) Werten für die Schrittweite, z.B.: 3, 8, 12.

 

do-while-Schleife mit Schrittweite

Implementieren Sie diese Schleife im Code-Rahmen als do-while-Schleife

Testen Sie diesen Code jeweils mit verschiedenen (kleinen) Werten für die Schrittweite, z.B.: 3, 8, 12.

Erinnerung: Sie können auch nur den selektierten Code ausführen
 
 

 

Fehlermöglichkeiten

Nun probieren wir, welche Fähler man beim Schreiben von Schleifen machen kann - damit Sie diese Fehler erkennen und beheben können.

Kopieren Sie dazu den Code der for-Schleife aus der vorherigen Aufgabe in einen Block-Kommentar, um diesen Fehlerfall vom Rest des Code-Rahmens zu trennen. 

Im Folgenden testen Sie in diesem Fehler-Block ein paar Fehlermöglichkeiten.

 

Zugriff auf Schleifenvariable nach der Schleife

Geben Sie testweise die Schleifenvariablen nach Ende des Körpers der for-Schleife, d.h. nach der schließenden Klammer } des Blocks, auf die Konsole aus.

Welchen Fehler meldet der Compiler nun - und warum?

Das ist ein weiteres Feature der Kontrollstruktur "Block"
 

 

Schleifenvariable heißt wie bestehende Variable

Definieren Sie nun vor der fehlerhaften for-Schleife eine Variable, die genauso heißt wie die Schleifenvariable.

Welchen Fehler meldet der Compiler nun - und warum?

 

Hilfs-Ausgaben für die Orientierung und Übersicht auf der Konsole

Allmählich werden die Konsole immer mehr Ausgaben angezeigt. Mit den folgenden Aufgaben produzieren sie immer weiter Ausgaben.

Tipp: Um die Übersicht darüber zu behalten, welche der geforderten Ausgaben zu welcher Aufgabe gehört, ist es eine gute Idee, die Überschrift der Aufgabe ebenfalls auf die Konsole auszugeben. Den Text dafür können Sie aus dem automatisch generierten Kommentar kopieren (dabei lassen Sie am besten die /// und den Pfeil aus).

Beispiel:

System.out.println("2.3 Fehlermöglichkeiten");

 

Verschachtelte Schleife

Geben Sie das Einmaleins von 1 bis n als Tabelle auf die Konsole aus.

Beispiel für das Einmaleins von 1 bis 3:

1    2    3
2    4    6
3    6    9

Dafür müssen Sie zwei Schleifen verwenden
 
Ordnen Sie die Zahlen möglichst geordnet untereinander an
 
 

Beschreiben Sie zunächst den Code für die Berechnung im Code-Rahmen und ergänzen Sie dann den Code dafür.

Testen Sie diesen Code dann mit kleinen Werten von n (3, 6, 9).

 

Einmaleins mit Überschriften

Erweitern Sie die Ausgabe des Einmaleins so, dass zusätzlich Überschriften für Zeilen und Spalten ausgegeben werden, z.B. so:

     1    2    3

1    1    2    3
2    2    4    6
3    3    6    9

Kopieren Sie den bisherigen Code für die Ausgabe der Tabelle und erweitern Sie ihn entsprechend.

 

Das Kopieren von Code ist immer mit Vorsicht zu genießen
