Einfache Arrays

Sie beginnen auf einfache Weise, den Umgang mit Arrays zu erlernen:

  • Sie erzeugen ein Array.
  • Sie füllen es mit Werten.
  • Sie geben die Werte wieder (auf die Konsole) aus.
  • Sie kopieren ein Array in ein anderes
  • Sie lernen einen typischen Laufzeitfehler kennen

 

Ein Array erzeugen

Wie gelernt, ist ein Array ein Referenz-Datentyp.

D.h. eine Array-Variable beinhaltet nicht das Array selber, d.h. nicht den Speicherplatz für die einzelnen Elemente, sondern "nur" eine Referenz auf diesen Speicherplatz. Der Speicherplatz für eine feste (!) Anzahl von Elementen des Arrays muss dann explizit zusätzlich erzeugt ("allokiert") werden.

Fehler beim Zugriff auf ein Array

Ein typischer Fehler beim Zugriff auf ein Array tritt nur zur Laufzeit auf und kann vom C-:ompiler nicht oder nur selten entdeckt werden: Sie dürfen für den indizierten Zugriff nur Indizes verwenden, für die auch ein allokierter (reservierter) Speicherplatz existiert.

Welche Indizes sind beim Zugriff auf die oben definierten Arrays erlaubt?
