Features und 5Coins

Was sind 5Coins?

In 5Code gibt es die "Währung" 5Coin: .

  • Sie können 5Coins auf mehrere Arten erhalten
  • Sie können 5Coins ausgeben und damit Features freischalten
  • Außerdem können Sie für 5Coins weitergehende Hilfen für die Lösung einer Aufgabe kaufen.

Was sind Features?

Unter "Features" werden hier einige Funktionen von 5Code verstanden, die das Editieren, Codieren, Programmieren, etc. unterstützen. Welche das im Einzelnen sind, wird gleich beschrieben. Sie werden hier besonders behandelt, weil einige Features nicht sofort zur Verfügung stehen, sondern im Verlaufe des Semesters nach und nach bereitgestellt werden. 

Der Grund dafür: Wenn Sie Programmier-Anfänger sind, sollen Sie nur mit den Funktionen umgehen müssen, die Sie auch wirklich brauchen. So können Sie sich immer auf Ihre eigentlichen Aufgaben konzentrieren.

Wenn Sie bereits Erfahrung in der Nutzung von IDEs haben, können Sie diese Features auch schon früher freischalten. Wie das geht, wird weiter unten beschrieben. 

Die Features sind geordnet nach den Bereichen Allgemein, Editor, Coding, Navigation, Information, Analyse.

Beschreibung von Features = Schnell-Hilfe

Beschreibung beim Freischalten

Schalten Sie nun in der Liste der Inaktiven Features das Feature "Textmarker" frei. Sie sollten genügend 5Coins dafür haben.

Sie bekommen dann, wie immer wenn Sie ein Feature freischalten (oder es automatisch freigeschaltet wird), eine Nachricht mit einer kurzen Erklärung des Features:

        

In diesem Dialog wird das Feature kurz beschrieben. 

  • Klicken Sie diesen Dialog also nicht sofort weg, sondern lesen Sie ihn kurz durch.

Die fortgeschrittene Nutzung des Textmarkers lernen Sie im Kapitel "Programmieren in 5Code" kennen. 

Beschreibung in der Feature-Liste

Die Beschreibung eines Features bekommen Sie ebenfalls jederzeit über die Feature-Liste: Klicken Sie in einer der oben aufgeführten Listen auf die Bezeichnung eines Features, dann wird eine kurze Beschreibung ausgeklappt.

       

Das funktioniert für alle, auch für noch nicht freigeschaltete Features.

Hilfe und ausführlichere Beschreibung

Mehr Hilfe und ausführlichere Beschreibungen von Features, Eigenschaften, etc. bekommen Sie natürlich auch über das Menü Hilfe.

 

5Coins erhalten

5Coins können Sie auf verschiedene Arten und Weisen erhalten, z.B. um damit Features zu erwerben (freizuschalten).

Durch Aktivitäten bei der Nutzung von 5Code

Wenn Sie Code compilieren (mit oder ohne Fehlermeldungen), Code ausführen, Kommentar schreiben, etc. können Sie sich 5Coins verdienen.

Durch Bearbeiten von Quizzes oder Aufgaben an anderer Stelle

Auch außerhalb von 5Code können 5Coins erworben werden:

  • Lösen von Aufgaben in der Vorlesung, im Praktikum, Seminar oder Tutorium
  • Lösen von Aufgaben in einer Lernumgebung wie Moodle

Ein 5Coin wird dabei als Gutscheincode angegeben, z.B.

U2FsdGVkX1+jsWkZKcGcGoGyfBPFO9gMJbourgOTD1BJrXNNnLbDM4Uiw57rnO6jLrGl3laeon9v8i/HvS0enw== 

Durch die Teilnahme an Evaluationen und Umfragen

Die Lehrveranstaltung und vor allem die Erfahrungen mit der Bearbeitung der Praktikumsaufgaben wird 1-2 mal im Semester evaluiert. Für die Teilnahme erhalten Sie ebenfals einen 5Coin-Gutschein.

Einlösen von Gutscheincodes für 5Coins

Sie sollten nun den oben angegebenen Gutscheincode einlösen.

Klicken Sie auf die Angabe Ihres "Kontostands" rechts unten am 5-Code-Window . Dann erscheint ein Dialog, in dem Sie den Text des oben angegebenen Gutscheincodes in die entsprechende Stelle einfügen und den Button klicken. Dadurch sollte sich Ihr Kontostand um 5 5Coins erhöht haben :-).

Jeder Gutscheincode kann nur einmal eingelöst werden, ist ja klar ;-)

 

Features kaufen für 5Coins

Wie kaufe ich ein Feature?

Für Ihre 5Coins können Sie sich Features "kaufen" und es damit freischalten. Wieviel ein Feature kostet, steht in der Liste (wenn Sie es noch nicht erworben haben):

Wenn Sie genügend 5Coins besitzen, können Sie dieses Feature freischalten (kaufen). Dazu klicken Sie auf die Zeile des Features, worauf u.a. ein Button erscheint, auf dem auch noch einmal die Kosten angegeben sind:

Wenn dieses Feature im Verlauf des Semesters für Sie kostenlos freigeschaltet werden wird, dann gibt es einen entsprechenden Hinweis.
 

Klicken Sie auf den Button [Freischalten für 5$]. Dann passiert folgendes:

  • Die Anzahl Ihrer 5Coins verringert sich um 5 
  • Das Feature wird freigeschaltet
  • Die Freischaltung des Features wird im Chat angezeigt

 

Features als Reaktion auf Ihre Aktivitäten

Features können auch als "Belohnung" oder als Reaktion auf Ihre Aktivitäten freigeschaltet werden. 

Wenn Sie z.B. häufiger mal einen Fehler beim Compilieren erzeugt haben, dann brauchen Sie etwas mehr Hilfe. Dann wird Ihnen - in einem folgenden Projekt - zusätzliche Unterstützung freigeschaltet.

Es können auch bereits Funktionen beim ersten Öffnen eines Projekts freigeschaltet werden. 

Lassen Sie sich überraschen :-).
