Concept Map ergänzen

Die Concept Map hatten Sie in der letzten Woche bereits heruntergeladen (Aufgabe _6_). Ein paar Begriffe hatten Sie auch schon auf die Zeichenfläche gezogen und miteinander verbunden. Diese Concept Map ergänzen Sie in jeder Woche um weitere Begriffe. Am Ende des Semesters haben Sie dann einen vollständigen Überblick über wichtige Begriffe des gesamten Semesters.

Öffnen Sie die Concept Map, dort finden Sie diese neuen Begriffe in der Liste am linken Rand. Die Concept Map öffnen Sie über das Menü Projekt -> Concept Map in neuem Fenster öffnen. 

Ziehen Sie die neuen Begriffe auf die Zeichenfläche

Verknüpfen Sie sie sinnvoll untereinander und beschreiben Sie jede Verknüpfung kurz.

Dabei wird nun unterschieden zwischen

  • Konzept-Begriffen, z.B. Kommentar, Datentyp, ...
  • Syntax-Begriffen, die Java mit einer bestimmten Schreibweise (Syntax) geschrieben werden können, z.B. Zeilenkommentar, Zeichen, ...

Konzept-Begriffe

Die Konzept-Begriffe stehen für sich selber, z.B. Wertebereich.

Sie können sie ggf. mit eigenen Notizen ergänzen und für sich erklären.

Syntax-Begriffe

Die "Syntax-Begriffe" stehen ebenfalls für ein Konzept. Es gibt aber zusätzlich eine oder mehrere Möglichkeiten, das Konzept in Java auszudrücken (zu schreiben). Es gibt zur Semantik (Bedeutung) des Konzepts also auch eine Syntax in der Programmierspracha Java.

Beispiel: Den Begriff "Zeilenkommentar" können Sie in der Map mit der Java-Syntax // ergänzen. Dazu schreiben Sie die Syntax in das weiße Feld direkt unterhalb des Begriffs. 

So können Sie das Konzept Zeilenkommentar, seine Semantik, mit seiner Umsetzung (Syntax) in Java verknüpfen. 

  • Das können Sie zum Nachschlagen der Syntax nutzen 
  • Und, noch wichtiger, Sie stellen im Kopf eine Verbindung zwischen Semantik und Syntax her

Notizen zu Begriffen

Ergänzen Sie gerne weitere Notizen zu jedem Begriff. Wie das geht, hatten Sie bereits in der letzten Woche gelernt. Sehen Sie ggf. im Hilfe-Menü (7.3.5) oder im Bot zum Thema Notizen nach.
