Gemeinsam arbeiten

Voraussetzungen

Sie können mit anderen gemeinsam arbeiten, sich im Chat unterhalten und Informationen direkt zum bearbeiteten Projekt auszutauschen, sowie gemeinsam an einem Projekt in Code-Dokumenten arbeiten.

Voraussetzungen

Dafür müssen diese Voraussetzungen erfüllt sein:

  • Sie sind mit dem Internet verbunden
  • Sie haben Ihre EMail-Adresse und einen Nutzernamen angegeben

Datenschutz und Netiquette

Die Kommunikation und die Zusammenarbeit mit anderen läuft über einen Server, der im Serverraum des FB Medien steht. Sämtliche ausgetauschten Informationen werden verschlüsselt ausgetauscht. Auf dem Server werden keine Nutzerdaten gespeichert, bis auf die, die für die technische Verbindung notwendig sind. Insbesondere werden keine Inhalte, z.B. von Chat-Posts, auf den Server gespeichert.

Alle Inhalte, die Sie über dieses Feature von 5Code an andere schicken, z.B. Chat-Posts, wird auf den Rechnern dieser anderen Teilnehmer*innen gespeichert. Ein nachträgliches Löschen einmal gesendeter Daten ist damit nicht möglich. Denken Sie also daran vernünftig, zivilisiert und höflich zu kommunizieren. Ein Verstoß gegen die IT-Richtlinien der HSD, denen Sie bei der Einschreibung zugestimmt haben, kann zum Ausschluss von der Nutzung aller IT-Dienste der Hochschule führen - incl. der Zusammenarbeit in 5Code.

Die Nutzung der Features zur Zusammenarbeit ist rein freiwillig. Sie können alle Aufgaben auch vollständig ohne diese Unterstützung der gemeinsamen Arbeit erfolgreich bearbeiten. Die Nutzung dieser Features stellt somit eine Einverständnis dar, dass Ihre Daten wie oben beschrieben verarbeitet werden. 

Kommunikation über Gruppen

Jede Kommunikation in 5Code mit anderen ist technisch immer eine Kommunikation in einer Gruppe - egal, wie viele Teilnehmer*innen gerade aktiv sind. 

Wenn Sie in der Sidebar den neuen Tab Collab/Chat öffnen, sehen Sie dort die Gruppen, die Ihnen bekannt sind:

Als Startpunkt ist für jeden 5Code-User eine persönliche Gruppe bereits vordefiniert. Diese hat den vom User beim ersten Start von 5Code angegebenen Usernamen.

Neue Gruppe definieren

Um eine weitere Gruppe hinzuzufügen, klicken Sie auf das [+] in der Überschrift "Meine Gruppen". Dann erscheint dieser Dialog: 

Um eine neue Gruppe zu definieren,

  • Klicken Sie auf den Button [Neu erzeugen].
    • Dann sehen Sie im Feld Gruppen-ID einen neuen Text, z.B. geben.gehirn.balkon
    • Wenn Ihnen die ID nicht gefällt, drücken Sie solange auf den Button, bis Sie eine passende ID sehen. 
    • Sie können NICHT selber irgendeine ID eingeben.
  • Geben Sie der Gruppe im nächsten Feld einen sprechenden Namen, z.B. Lerngruppe Rookies.
    • Dieser Name wird dann in der oben beschriebenen Liste der bekannten Gruppen angezeigt.
  • Speichern Sie dann diese Angaben

Danach erscheint die neue Gruppe in der Liste Ihrer Gruppen:

Andere in eine Gruppe einladen

Wenn Sie andere in diese Gruppe einladen möchten, geben Sie Ihnen die Gruppen-ID dieser Gruppe weiter (geben.gehirn.balkon), nicht aber den Namen. Nur über die ID können Gruppen eindeutig identifiziert werden.

Wie Sie die Gruppen-ID einer Gruppe in Ihrer Liste herausfinden, steht weiter unten.

Einer bestehenden Gruppe beitreten

Analog können Sie einer bestehenden Gruppe beitreten, wenn Sie ihre Gruppen-ID bekommen haben. Öffnen Sie dazu den gleichen Dialog über den [+] Button.

  • Jetzt geben Sie im Feld Gruppen-ID die ID ein, die Sie bekommen haben.
  • Geben Sie der Gruppe ggf. einen anderen Namen, der für Sie passend ist
  • Speichern Sie die Angaben

Nun erscheint diese Gruppe ebenfalls in der Liste Ihrer Gruppen. 

Eine Gruppe für die Kommunikation auswählen und chatten

Klicken Sie in der Liste "Meine Gruppen" eine Gruppe an, in der Sie mit anderen kommunizieren möchten. Dann sieht die Sidebar z.B. so aus:

In der Überschrift steht nun der Name der Gruppe.

Darunter können Sie die Liste der aktiven User aufklappen, die gerade in diesem Moment ebenfalls in dieser Gruppe online sind. 

Unter diesen Informationen wird der gerade laufende Chat angezeigt. Wenn Sie etwas posten möchten, schreiben Sie das in das Eingabefeld unten in der Sidebar - genauso, wie Sie das von anderen Messengern kennen. 

Wenn Sie die Kommunikation in dieser Gruppe verlassen wollen, klicken Sie auf den [<-] Button oben links. 

Gruppen-ID einer Gruppe herausfinden

Wählen Sie in der Liste "Meine Gruppen" eine Gruppe aus. Klicken Sie dann auf den Namen der Gruppe in der Überschrift, dann wird der oben bereits beschriebene Dialog angezeigt. Die nun gezeigte Gruppen-ID können Sie weitergeben, wie oben ebenfalls bereits erklärt.

Chat mit anderen

Im Chat können Sie mit beliebig vielen anderen Leuten Text-Nachrichten austauschen. Dafür könnten Sie auch andere Messenger-Dienste wie WhatsApp verwenden, dafür müssen Sie aber immer wieder das Programm wechseln.

Außerdem können Sie sich so unterhalten, ohne ihre Mobilfunk-Nummer preisgeben zu müssen.

Weiterhin können Sie (demnächst) auch Links in den Code oder die Aufgabe verschicken, die alle anderen verwenden können, die das gleiche Projekt bearbeiten. 

Wie Sie mit anderen chatten, ist bereits oben beschrieben.

Chat-Posts werden nur empfangen, wenn Sie online dabei sind

Um das Chat-System so einfach wie möglich zu machen, und gleichzeitig so wenig Nutzerdaten auf dem Kommunikations-Server zu speichern, werden keine Nutzerdaten auf dem Server gespeichert. 

D.h. es werden auch keine Posts zwischengespeichert, die gesendet werden, während Sie gerade nicht online sind.

Alle empfangenen Chat-Posts werden auf Ihrem Rechner gespeichert.
