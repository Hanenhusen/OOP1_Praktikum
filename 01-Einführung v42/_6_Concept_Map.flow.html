Concept Map

In der OOP1-Vorlesung begegnen Sie vielen neuen Begriffen zum Thema Java und Programmieren allgemein. Um sich ihre Bedeutung und ihre Zusammenhänge untereinander einfacher zu merken, kann eine sogenannte Concept Map sehr gut helfen.

Was ist das und was nutzt es? 

Was ist eine Concept Map?

Eine Concept Map visualisiert Begriffe und ihre Zusammenhänge. Verbindungen / Zusammenhänge zwischen Begriffen werden außerdem näher beschrieben, wie z.B. "ist ein" oder "ist ein Teil von" oder "wird häufig verwendet für". Die Verbindungen laufen von einem Begriff zu einem anderen, sie werden also als Pfeil dargestellt.

Vielleicht kennen Sie schon eine Mind Map? Eine Concept Map ist sehr ähnlich, sie hat aber keinen zentralen Begriff und ist nicht als Baum aufgebaut. Vielmehr können Begriffe untereinander beliebig miteinander verknüpft werden.

Beispiel:

Sie bauen schrittweise eine OOP1-Concept Map auf

Für OOP1 bauen Sie schrittweise eine Concept Map auf. In jeder Woche kommen mit jedem neuen Aufgaben-Projekt neue Begriffe hinzu. Sie ergänzen und beschreiben dann ihre Beziehungen untereinander.

  • Damit wiederholen Sie sie jede Woche noch einmal und machen sich ihre Verbindungen untereinander klar.  So festigen Sie jede Woche Ihr Wissen.
  • Außerdem können Sie hier eintragen, wo Sie noch Fragen haben, oder Erklärungen ergänzen. 
  • Am Ende des Semesters habe Sie eine komplette Übersicht aller wichtigen Begriffe und ihrer Beziehungen untereinander aus OOP1. Das ist zur Prüfungsvorbereitung ebenfalls sehr hilfreich.

 

Arbeiten mit einer Concept Map

Woher kommen die Begriffe?

Die Begriffe, die Sie auf der Map verknüpfen sollen, müssen Sie nicht selbst eingeben. Sie werden Ihnen im Verlauf des Semesters innerhalb eines Aufgaben-Projekts zur Verfügung gestellt: Immer, wenn ein Aufgaben-Projekt zum ersten Mal geöffnet wird, wird die Map um neue Begriffe ergänzt. So arbeiten Sie immer mit den Begriffen, die Sie bereits aus der Vorlesung kennen und die relevant für die aktuelle Aufgabe sind.

Verbinden von Begriffen

Um zwei Begriffe miteinander verbinden zu können, müssen beide auf der Zeichenfläche liegen. Beim Hovern mit der Maus über einen Begriff sehen Sie grüne Ankerpunkte für Verbindungen. Durch Klick-Ziehen-Loslassen können Sie eine solche Verbindung ziehen.

Der so entstandene Pfeil ist gestrichelt, da die Verbindung noch nicht vollständig ist: In einer Concept Map muss jede Verbindung kurz beschrieben werden. Dieser kurze Text kann durch Doppelklick auf den Pfeil hinzugefügt werden.

Weitere Bearbeitung 

Mehr Platz schaffen

Die Zeichenfläche erweitert sich automatisch, wenn Sie Begriffe außerhalb des karierten Bereichs schieben.

Sie können auch mehrere Begriffe auswählen (mit Shift-Click oder durch Aufziehen eines Rechtecks) und verschieben, um mehr Platz in einem Bereich zu schaffen.

Export als Bild

Über den Button kann die gesamte Map als Bild exportiert werden. So können Sie sie z.B. auch ausdrucken.

Ordnen von Begriffen mit Ort und Farbe

Wenn immer mehr Begriffe im Verlaufe des Semesters hinzukommen, lohnt es sich, diese weiter zu ordnen und zu strukturieren. Dazu haben Sie zwei Möglichkeiten:

  • Sie gruppieren Begriffe örtlich, d.h. sie schieben sie näher zusammen
  • Sie gruppieren Begriffe durch verschiedene Farben

Beim Klick auf einen Begriff öffnet sich ein Pop-Up-Menü, mit dem Sie die Farbe des Elements nach Belieben ändern können (Paletten-Icon). 

Sowohl das Verschieben als auch das Setzen einer Farbe funktioniert auch für eine Gruppe von mehreren ausgewählten Begriffen.

Markieren von Begriffen mit Badges

Sie können jeden Begriff auch mit einem "Ampelsystem" markieren:

  • grün:  Der Begriff ist mir klar
  • gelb:   Hier müsste ich noch mal überlegen oder etwas nachschlagen
  • rot:     Der Begriff ist mir völlig unklar, hier müsste ich eine Frage stellen

Dazu finden Sie im gleichen Pop-Up-Menü (Figuren-Icon) drei Badges, die Sie an den Begriff heften können. 

Eigene Notizen ergänzen

Außerdem können Sie sich zu jedem Begriff Notizen machen:

  • Eigene Erklärungen
  • Verweise auf das Skript 
  • Fragen, die noch geklärt werden müssen 

Wie Sie diese Notizen machen können, wird allgemein im nächsten Kapitel Bot_Notizen erläutert.

 

Konzept-Begriffe und Syntax-Begriffe

Bei den Begriffen in der Concept-Map unterscheiden wir noch zwischen "Konzept-Begriffen" und "Syntax-Begriffen" - aus praktischen Gründen, wie wie gleich sehen werden. 

Konzept-Begriffe

Die "Konzept-Begriffe" stehen für ein eher allgemeines Konzept, z.B. Wertebereich, Kommentar, Datentyp, Software, ...

Sie können Konzept-Begriffe in der Concept-Map mit eigenen Notizen ergänzen. Wie Sie diese Notizen machen können, wird allgemein im nächsten Kapitel Bot_Notizen erläutert.

Syntax-Begriffe

Die "Syntax-Begriffe" stehen für ein konkretes Konzept. Es gibt für diese zusätzlich eine oder mehrere Möglichkeiten, dieses konkrete Konzept in Java auszudrücken (zu schreiben). Es gibt zur Semantik (Bedeutung) des Konzepts also auch eine Syntax - daher die Bezeichnung "Syntax-Begriff". Damit Sie diese Syntax auch direkt beim Begriff aufschreiben können, hat jedes Element für einen Syntax-Begriff ein zusätzliches weißes Feld direkt unterhalb des Begriffs. Dorthinein können Sie dann den Java-Code eintragen. 

Beispiel: Den Begriff "Zeilenkommentar" können Sie mit der Java-Syntax // ergänzen. 

So können Sie das Konzept Zeilenkommentar, seine Semantik, mit seiner Umsetzung (Syntax) in Java verknüpfen.

Das können Sie zum Nachschlagen der Syntax nutzen.
Und, noch wichtiger, Sie selber stellen im Kopf eine Verbindung zwischen Semantik und Syntax her.

Sie können natürlich auch Syntax-Begriffe in der Concept-Map mit eigenen Notizen ergänzen. Wie Sie diese Notizen machen können, wird allgemein im nächsten Kapitel Bot_Notizen erläutert.

Syntax-Begriffe fügen Sie in der nächsten Woche zur Concept Map hinzu.
